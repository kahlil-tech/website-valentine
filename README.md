# website-valentine
Website ini aku buat khusus untuk merayakan Hari Valentine dengan cara yang berbeda. Bukan sekadar kata-kata, tapi ruang digital yang berisi perasaan, kenangan, dan harapan yang tulus. Setiap bagian dibuat dengan cinta, sebagai simbol perhatian dan keseriusan dalam menghargai seseorang yang spesial.
